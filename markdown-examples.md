# 我的计算机之路

## 填志愿
曾经的我对计算机这个行业懵懵懂懂，完全不懂其中的奥秘。高中的时候，刷视频偶然看到别人在终端上运行某个指令，随后就出现了一串串的字符串流，那一刻觉得实在是太牛逼了！

高考结束后，看着那寒酸的分数，我完全不知道该怎么填志愿。这时候，有个引路人就显得无比重要。很幸运，我遇到了一位大一届的学长，他正好是学计算机的。和他交流下，我也选择了计算机专业。
